#  Rota Certa

O **Rota Certa** é uma ferramenta web para entregadores, focada em simplicidade e eficiência. Ele permite organizar uma lista de entregas diárias e visualizar todos os destinos em um mapa com um único clique.

---

##  Funcionalidades principais

* **Adição Manual:** Digite o endereço e adicione à lista rapidamente.
* **Leitor de QR Code:** Use a câmera do celular para ler endereços em etiquetas de pacotes.
* **Mapa Blackout:** Visualização noturna (fundo preto) para economizar bateria e facilitar a visão em ambientes externos.
* **Contador de Paradas:** Controle em tempo real de quantas entregas restam no dia.
* **Armazenamento Local:** A lista não se apaga se você fechar o navegador.
* **Limpeza de Jornada:** Botão para resetar todos os dados ao final do expediente.

---

## 🚀 Tecnologias e Ferramentas

Este projeto utiliza tecnologias leves para garantir que rode bem em qualquer celular:

* **Leaflet.js** (Motor do Mapa)
* **OpenStreetMap** (Dados Geográficos)
* **Html5-QRCode** (Tecnologia de Scanner)
* **JavaScript/HTML5/CSS3** (Estrutura e Lógica)

## 📋 Como usar

1. Copie o código do arquivo `index.html` para o seu servidor ou computador.
2. Certifique-se de acessar via **HTTPS** para que o leitor de QR Code funcione.
3. Adicione suas paradas e clique em **"Otimizar Rota no Mapa"**.
4. Use o botão **"Resetar Jornada"** para começar um novo dia.
